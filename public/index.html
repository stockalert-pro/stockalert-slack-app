Installation Guide

Getting Started

  1. Add to Slack: Click the "Add to Slack" button above to install the app in your workspace
  2. Choose a Channel: Select which channel should receive stock alerts
  3. Get Your API Key: Log in to your StockAlert.pro account and copy your API key from the settings page
  4. Connect Your API Key: In Slack, run /stockalert apikey sk_your_api_key
  5. Verify Setup: Run /stockalert status to see your webhook URL and connection status
  6. Test Connection: Run /stockalert test to send a test alert

Slash Commands

Use these commands in any channel where the bot is present:

  • /stockalert help - Show available commands
  • /stockalert status - Show integration status and webhook URL
  • /stockalert test - Send a test alert to the channel
  • /stockalert channel #channel-name - Set default alert channel
  • /stockalert apikey sk_... - Connect StockAlert.pro API key
  • /stockalert disconnect - Remove API connection

Requirements

  • An active StockAlert.pro account
  • Slack workspace admin permissions (for installation)
  • A public or private channel for receiving alerts
